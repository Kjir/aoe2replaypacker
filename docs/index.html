AoE2 Replay Pack Generator

This page will help you create a zip file of replays for submitting your match results. To use it follow, these simple steps:

  1. Fill in the player names
  2. For each replay, click the "choose file" button and navigate to your replay
    1. Go to "%USERPROFILE%\Games\Age of Empires 2 DE"
    2. Click the folder with a bunch of numbers
    3. Go to the "savegame" folder
  3. Once all replays are selected, click download

Note: if the match ended 2-0, simply leave the third file empty and a dummy file will automatically be generated and used

Technical note: All of this is done in your browser, no data is uploaded anywhere.
